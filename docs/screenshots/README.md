# Screenshots

Drop your desktop captures here with these filenames so the main `README.md` picks them up:

| File | Suggested content |
|---|---|
| `desktop.png`     | Clean Hyprland desktop with Waybar and wallpaper visible |
| `claude.png`      | The Claude popup mid-conversation, Waybar Claude module showing |
| `cheatsheet.png`  | The `SUPER+SHIFT+K` cheatsheet open with a search filter applied |
| `lock.png`        | `hyprlock` screen with the time / date / input field visible |
| `wallpaper.gif`   | A short loop showing the swww transition + matugen recolor cycle |
| `terminal.png`    | A new kitty terminal with the random Pokémon + fastfetch greeting |

Recommended capture resolution: **1920×1080** or higher, lossless PNG.

Quick capture command (full output): `hyprshot -m output -o ~/Repositories/dotfiles/docs/screenshots/`
